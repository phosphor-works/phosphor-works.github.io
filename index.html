Phosphor

A Qt6 / Wayland library suite for window-management tools.

Zones, layers, layout algorithms, shader effects, global shortcuts — packaged as focused LGPL libraries. Built for KDE Plasma and reusable by any Wayland compositor or Qt host.

The Suite

Ten libraries, each with a single responsibility. Docs pages below are still being written — everything currently lives inside the PlasmaZones repository.

  • phosphor-shell Wayland layer-shell & QPA plugin for overlays
  • phosphor-identity Stable window identity primitives (WindowId)
  • phosphor-rendering ShaderItem / RenderNode infrastructure
  • phosphor-animation Window motion curves & animation controllers
  • phosphor-zones Zone data model & persistence (JSON)
  • phosphor-tiles Tiling algorithms — built-in + scripted JS
  • phosphor-shortcuts Pluggable global-shortcut backends (KGlobalAccel / XDG / D-Bus)
  • phosphor-layer Layer-based overlay rendering
  • phosphor-layout-api Layout description interfaces
  • phosphor-config Pluggable configuration backends
