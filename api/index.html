Not built yet?

The /api/html/ tree is gitignored — it's regenerated from source by a Doxygen run, so the first time you preview this site locally the links above will 404 until you build the reference.

Install prerequisites (doxygen + graphviz) and run the build script:

# Arch / CachyOS:
sudo pacman -S doxygen graphviz
# Debian / Ubuntu:
sudo apt install doxygen graphviz
# Fedora:
sudo dnf install doxygen graphviz

# From the repo root:
./scripts/build-docs.sh
# ...or to wipe the generated tree first and auto-open:
./scripts/build-docs.sh --clean --open

Output lands at api/html/index.html. By default the script pulls headers from ../PlasmaZones3/libs/phosphor-*/include/; override with PHOSPHOR_SRC=/path/to/PlasmaZones3 if your checkout lives elsewhere.
